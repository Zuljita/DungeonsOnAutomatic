Environmental details like lighting, ceilings, mana, sanctity, and nature's strength can now be imported via the Environment module using CSV files with name, description, and weight columns. For quick overrides, you can still upload JSON files named custom-lighting.json, custom-ceilings.json, custom-dfrpg-mana-levels.json, custom-dfrpg-sanctity-levels.json, and custom-dfrpg-natures-strength.json.
